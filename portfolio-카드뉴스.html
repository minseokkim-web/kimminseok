카드뉴스 디자인 프로젝트

"한 장씩 넘기며 이해되는, 가볍고 명확한 메시지"

Project information

  • 작품명: 정보형 카드뉴스 디자인 시리즈
  • 형식: SNS 콘텐츠용 카드뉴스 (정사각형/세로형)
  • 컨셉: 짧고 명확한 메시지 + 시각적 집중도 높은 레이아웃
  • 타깃: 10~30대 SNS 사용자 / 모바일 중심 콘텐츠 소비층
  • 톤앤매너: 심플, 직관, 포인트 컬러 강조, 모바일 최적화
  • 컬러: 화이트, 블랙, 민트, 옐로우 등 상황별 변형

설명

본 프로젝트는 정보 전달과 시선을 동시에 사로잡을 수 있는 카드뉴스 콘텐츠를 기획하고 디자인한 시리즈입니다. 콘텐츠 소비 흐름에 맞춰 각 장마다 핵심 키워드를 분산 배치하고, 사용자의 시선을 유도할 수 있도록 컬러, 여백, 타이포를 세심하게 구성했습니다.

기획 의도

정보 콘텐츠는 너무 많은 텍스트로 인해 사용자 피로도를 높이기 쉽습니다. 그래서 카드뉴스는 '한 장당 하나의 메시지'를 원칙으로 구성해 **가볍게 넘기면서도 내용이 확실히 남도록 디자인했습니다. 특히 SNS 환경에서의 탐색성, 공유성, 가독성을 중심에 두고 실용성과 시각적 흥미 사이의 균형을 맞추는 데 집중했습니다.
