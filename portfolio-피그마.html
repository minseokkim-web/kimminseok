키키

빠른 응답, 명확한 정보, 커뮤니티 참여를 위한 UX 구조

프로젝트 개요

  • 작업명: : 실종 반려동물 커뮤니티 앱 ‘키키(KIKI)’
  • 작업 영역: 기획 / UX 설계 / 와이어프레임(Figma) / 콘텐츠 구성
  • 작업 목적: 실종 반려동물 문제를 하이퍼로컬 커뮤니티 기반으로 실질적 해결

핵심 기능 요약

  • BLE + GPS 연동 기반, 반경 1km 이내 사용자에게 실시간 푸시 전송
  • 사진/시간/장소 태그와 함께 간편 제보
  • 하이퍼로컬 지도 기반의 ‘실종/제보 현황’ 시각화
  • 실종 전 대비를 위한 사용자 맞춤 데이터 저장
  • 소액 보상금 또는 후원 구조로 자발적 제보 유도

    • 기획 의도

    단순 위치 추적 기능을 넘어서, 실시간 알림과 지역 커뮤니티의 자발적 참여로 잃어버린 반려동물을 찾을 수 있는 플랫폼을 만들고 싶었습니다. 반려인의 불안과 사회적 무관심 사이를 연결하는 디지털 감성 구조 요청 시스템을 목표로 기획했습니다.
