POST MALONE WORLD TOUR

"콘서트 홍보 포스터 디자인"

Project information

  • 작품명: POST MALONE WORLD TOUR 콘서트 포스터
  • 컨셉: 강렬한 흑백 톤 + 연기 효과를 통한 몰입감 있는 무대 연출
  • 타깃: 공연 마케팅 관계자, 20~30대 음악 팬, 브랜드 아트워크 관심층
  • 톤앤매너: 강렬함, 몰입, 무게감, 도시적 감성
  • 컬러: 블랙, 화이트, 스모크 그레이, 라이트 실버

설명

이 포스터는 세계적인 아티스트 Post Malone의 2025 월드 투어 서울 공연을 가상으로 기획해 제작된 콘서트 홍보용
메인 비주얼 디자인입니다. 무대 조명, 아티스트의 시그니처 감성, 강렬한 흑백 톤을 중심으로 공연의 에너지와 몰입감을 시각적으로 강조했습니다.

기획 의도

평소 Post Malone의 열렬한 팬으로, 첫 내한 공연도 직접
관람했지만 그날의 티켓과 포스터가 아쉬워서, 아티스트의 무드와 무게감을 담아 제가 직접 디자인하게 되었습니다.
강렬한 흑백 대비와 연기 효과를 활용해, 실제 무대의 생동감과 브랜드 이미지를 시각화했습니다.
